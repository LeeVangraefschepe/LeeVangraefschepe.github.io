Short description

For my bachelor's thesis, I will conduct benchmarking and testing of various methods for bridging cross-language networking. This testing will assess latency, bandwidth, and CPU/Memory usage. To visualize these findings, I will use a Minecraft Server that sends data to a C++ client.
