Tron battle tanks

Creation date: 22 february 2023

Short description

Tron battle tanks is fully made in a self-written engine in the language C++. It uses SDL for rendering and sound, ImGui for debugging purpose and glm for math and vectors. It currently has over 1000 views on github and close to 1000 clones.

Game engine features

  • Scene management
  • Sprite and text rendering
  • Subject & observer
  • GameObjects
  • Multithreaded audio
  • Overlapping physics
  • Command system
  • Controller & keyboard input

Used patterns

  • Game loop
  • Component (composition)
  • Update method (destoying objects in update works)
  • Dirty flag
  • Singleton & Service locator
  • Command
  • Observer
  • Mutlithreading (locks & queue)
